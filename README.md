# TSP-Parallel-Genetic-Algorithm-in-Scala
Implementation of Parallel Genetic Algorithm in Scala for solving Travelling Salesman Problem.

# To run genetic algorithm:
`position in sources`<br>
`sbt`<br>
`compile`<br>
`run`<br>
